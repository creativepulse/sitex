Sitex (Simple Text) outputs plain text content on the front-end without any filtering.

== Description ==

Sitex outputs plain text in HTML pages without any filtering. It is ideal when
you want to output raw HTML or JavaScript code and you want it to appear exactly
as you set it without any filtering.

== Installation ==

Use the regular installation procedure to install this plugin.

That is, go to Admin > Plugins > Add New > Upload.

Alternatively you can upload it into the `/wordpress/wp-content/plugins/` directory
and then activate it in the Admin control panel for plugins.

== Example usage ==

Example 1:

    <p>Paragraph with a word in <strong>bold</strong>
    <br/>and a word in <span style="color:red">red color</span></p>

Example 2:

    <script type="text/javascript">
    document.writeln('<p>The URL is ' + window.location + '</p>');
    </script>
